# Article Display Standards

## Display Goals
- Make the abstract and metadata visible immediately.
- Provide frictionless access to downloads.
- Keep the reader oriented to the journal context.

## Required Metadata
- Title
- Authors + affiliation
- Journal title
- Publication date
- DOI
- Keywords

## Article Card (List View)
- Title + author line
- Abstract snippet (2–4 lines)
- Keyword chips
- DOI line
- “Read Article” or “Download” CTA

## Article Detail Page
### Above the fold
- Journal label
- Title
- Abstract
- DOI
- Author line

### Access
- If PDF available → Preview + Download
- If DOCX only → Download only (preview disabled)

### Supplemental
- Keywords list
- Academic context paragraph
- Journal information sidebar

## Visual Rules
- Use consistent spacing (padding 24–32px).
- Avoid multi-column metadata blocks on mobile.
- Ensure contrast meets accessibility thresholds.

## Editorial Consistency
- All dates formatted as YYYY-MM-DD.
- DOI displayed as `DOI: 10.xxxx/...`
- Use consistent journal naming with “NewTIFI” prefix.
